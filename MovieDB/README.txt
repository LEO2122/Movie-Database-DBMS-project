---------------------------------------MOVIE DATABASE---------------------------------------


Installation :

* Install the latest XAMPP version from https://www.apachefriends.org/download.html


Configuration :

* Copy the mdb folder to (drive):\xampp\htdocs directory.

* Run XAMPP control and start Apache and MySql server.

* Import the moviedb file in phpMyAdmin using the import function.

Enter http://localhost/mdb/main.php in the browser.
